## CodersHub

### 简介

CodersHub 是一个基于 Jetpack Compose 实现的关于程序员类型的社交网络应用程序（学习用途）

使用到的技术栈：

* **语言**：Kotlin
* **UI框架**：Jetpack Compose + MVI
* **依赖注入**：Dagger Hilt
* **网络库**：OKHttp、Retrofilt、Scarlet
* **图片加载库**：Coil、uCrop
